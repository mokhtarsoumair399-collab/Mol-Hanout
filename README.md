## 🇬🇧 English

# Mol Hanout

An Expo app to track customer debts inside a shop, featuring an Arabic UI, RTL support, and WhatsApp reminders.

## 🚀 Run

```bash
nvm use
npm install
npx expo start
eas build -p android --profile preview
```

## ⚙️ Environment Requirements

* Uses `Expo SDK 54`
* Requires `Node 20.19.4` or newer
* `Node 18` may cause errors (`toReversed is not a function`)

## 📦 Build Android & iOS

```bash
npm install
npx eas login
npx eas build -p android --profile production
npx eas build -p ios --profile production
```

* Config: `eas.json`
* Android package: `com.molhanout.app`
* iOS bundle: `com.molhanout.app`

## ✨ Features

* Customer management (add, edit, delete, search) 
* Debt & payment tracking with automatic balance
* Dashboard (total debts, customers count, top debtors)
* Customer detail with transaction history
* WhatsApp reminders
* Bulk WhatsApp messaging
* Auto-message system (reminders, follow-ups, confirmations)
* Inventory management (stock tracking + alerts)
* Backup & Restore option
* Daily local notifications per customer
* Reminder scheduling with time picker
* Instant test notification
* Toast feedback system
* Offline-first (`AsyncStorage`)
